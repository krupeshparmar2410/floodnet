Under the Hood

The Models Powering This

Two purpose-built machine learning pipelines — each fine-tuned on the FloodNet aerial imagery dataset and optimised for real-world deployment.

Classification

Flood/Non-Flood Classifier

A Random Forest classifier trained on HOG texture descriptors and 3-channel colour histograms. Resizes input to 128×128 and outputs a binary verdict with a calibrated probability score.

  • HOG features (9 orientations, 16×16 cells)
  • 32-bin colour histograms per channel
  • Outputs: Flooded / Non-Flooded + confidence %
  • Inference in < 2 seconds on CPU
Segmentation

Pixel-Wise Semantic Segmenter

Patch-based binary classifier mapped to 10 semantic classes via a colour-profile heuristic. Slides a 32×32 window with 16-pixel stride across a 256×256 image and blends the resulting label map onto the original.

  • 10 classes: Water, Buildings, Roads, Trees, Grass…
  • Colour-based sub-classification per patch
  • Uncertainty detection via confidence spread
  • Output: blended overlay + area distribution chart

Model Architecture

Classification & Segmentation

🌲

Task 1 — Image Classification

Random Forest classifier trained on HOG texture features combined with multi-channel colour histograms. Images are resized to 128×128 and converted to grayscale for HOG extraction. The colour component uses 32-bin histograms over each BGR channel. The final feature vector is ~1,100-dimensional. Outputs a binary label (Flooded / Non-Flooded) with a calibrated confidence percentage derived from predict_proba.

🌊

Task 2 — Semantic Segmentation

Patch-based sliding-window approach over 256×256 resized images. A 32×32 window slides with stride 16, producing a binary flood/non-flood prediction per patch. A colour-profile heuristic then refines each prediction into one of 10 FloodNet semantic classes (Water, Building-Flooded, Road, Tree, Grass, Vehicle, Pool…). Class labels are colour-coded, blended 40/60 with the original image, and returned as a Base64-encoded PNG alongside per-class percentage distributions.

Vision

Purpose & Future Scope

This project demonstrates that classical machine learning — without deep learning — can produce actionable flood intelligence from aerial imagery.

🎯 Current Purpose
  • Rapid flood/non-flood classification of aerial images
  • Pixel-level semantic mapping for spatial damage assessment
  • Confidence-aware uncertain predictions to avoid false certainty
  • Lightweight CPU deployment — no GPU infrastructure needed
  • REST API integration ready for embedding in wider platforms
🚀 Future Scope
  • Deep learning (U-Net, DeepLab) for pixel-accurate segmentation
  • Real-time satellite image ingestion pipeline
  • Multi-temporal flood progression tracking
  • GIS integration for geospatial map overlays
  • Mobile-first field app for responders with offline support
